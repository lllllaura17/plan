# plan
Планирование активностей представляет собой инструмент для оценки эффективности, безопасности и целесообразности применения различных способов передвижения (пешего, колесного, специального) в зависимости от специфики ландшафта (горная среда, пересеченная или лесная местность) и поставленных задач.
